# Commandes d'interfaçage

## Menu :

```
______________________________________________________________________________

- cd <répertoire> : Permet de se déplacer dans l'arborescence des fichiers.
    Exemple : 
        cd monDossier
    Infos : Pour revenir en arrière : cd ..
______________________________________________________________________________

- dir             : Liste tous les fichiers et dossiers dans un répertoire.
    Exemple : 
        C:\Users\a> dir 
            15/02/2020  18:26    <DIR>          .
            15/02/2020  18:26    <DIR>          ..
            15/02/2020  19:15    <DIR>          Desktop
            13/02/2020  21:00    <DIR>          Documents
            15/02/2020  18:34    <DIR>          Downloads
            13/02/2020  21:00    <DIR>          Music
            15/02/2020  18:26    <DIR>          Videos
                   2 fichier(s)              444 octets
                  6 Rép(s)  70 533 074 944 octets libres
______________________________________________________________________________

- call            : Exécute un programme.
    Exemple :
        C:\Users\a> call aProgram.exe

```
